# Intro to Webmapping: Part 1 - Online Platforms
### UBC Library Research Commons
  

Description: This workshop provides a gentle introduction to web mapping using two free browser-based platforms: [uMap](https://umap.openstreetmap.fr/en/) and [Google MyMaps](https://www.google.com/maps/about/mymaps/).Join us to learn how to dynamically and interactively visualize geospatial data with maps that can be shared over the web.
